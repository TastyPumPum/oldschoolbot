OSB has several GP gambling modes under [[/gamble]].

## Modes

| Mode       | Command Example                         | Bet Range |
| ---------- | --------------------------------------- | --------- |
| Blackjack  | [[/gamble blackjack amount\:100m]]      | 1m - 500m |
| Dice       | [[/gamble dice amount\:100m]]           | 1m - 500m |
| Slots      | [[/gamble slots amount\:100m]]          | 20m - 1b  |
| Lucky Pick | [[/gamble lucky_pick amount\:100m]]     | 1m - 3b   |

## Detailed Pages

- [Blackjack](/osb/miscellaneous/blackjack/)
- [Dice](/osb/miscellaneous/dice/)
- [Slots](/osb/miscellaneous/slots/)
- [Lucky Pick](/osb/miscellaneous/lucky-pick/)

## Notes

- Ironmen cannot use GP gambling modes.
- If you use self-gambling lock, GP gambling commands are blocked.

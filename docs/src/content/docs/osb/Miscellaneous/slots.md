Slots is a 3x3 board gamble with symbol multipliers.

Start with:
[[/gamble slots amount\:100m]]

## Bet Limits

- Minimum bet: 20m GP
- Maximum bet: 1b GP

## Rules

- You must confirm the bet before spinning.
- A win requires **3 matching symbols in a horizontal row**.
- If multiple rows win, the best multiplier is used.
- If no row matches, you lose the full bet.

## Multipliers

- Scroll: 2x
- Dinhs: 3x
- DClaws: 5x
- TBow: 15x
